# 特徴
- シンプルな仕組みで、作業内容をカテゴリとして登録し、メイン画面からスタート/ストップの操作によって作業時間を記録できる。
- 日付をクリックすることで、詳細作業内容（作業時間、同日の他の作業内容、同日の合計作業時間）を確認できる。
- 作業時間をチャート化することで、月別の作業を視覚化できる。（未実装）
